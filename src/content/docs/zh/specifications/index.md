## 主要章节

C2PA 规范涵盖：

### 1. 核心概念
- 清单和声明
- 断言和证据
- 加密签名
- 信任模型

### 2. 技术实现
- 文件格式集成
- 元数据结构
- 签名和验证
- 证书要求

### 3. 支持的媒体类型
- 图片（JPEG、PNG、WebP、AVIF）
- 视频（MP4、MOV、WebM）
- 音频（MP3、WAV、AAC）
- 文档（PDF - 进行中）

### 4. 高级功能
- 成分关系
- 编辑和隐私
- 硬绑定
- 软绑定
